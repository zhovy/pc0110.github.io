# 下面是 Linux 系统定不同类型文件默认的颜色：

  • 白色：表示普通文件
  • 蓝色：表示目录
  • 绿色：表示可执行文件
  • 红色：表示压缩文件
  • 浅蓝色：链接文件
  • 红色闪烁：表示链接的文件有问题
  • 黄色：表示设备文件
  • 灰色：表示其它文件

# 常见的使用命令

  1. top rm cd cp
  2. ps 、pstree、 ps -ef|grep java
  3. free cat chmod ls find
  4. touch tail

# 文件操作

  1. 创建文件
    touch 文件名
  2. 删除文件
    rm 文件名
  3. 移动文件
    mv 文件名 目标地址
  4. 复制文件
    cp 文件名 目标地址
  5. 重命名文件
    mv 文件名 目标地址
  6. 切换目录
    cd 目录名
  7. 创建目录
    mkdir 目录名
  8. 删除目录
    rm -rf 目录名
  9. 切换用户
    su 用户名
  10. 切换到根目录
    cd /
  11. 切换到用户目录
    cd ~
  12. 切换到上一次目录
    cd -

# 查看系统资源

  1. 查看 cpu 信息
    cat /proc/cpuinfo
  2. 查看内存信息
    cat /proc/meminfo
  3. 查看磁盘信息
    df -h
  4. 查看网络信息
    ifconfig
  5. 查看进程信息
    ps -ef|grep java
  6. 查看端口信息
    netstat -anp|grep 端口号
  7. 查看系统信息
    uname -a
  8. 查看系统时间
    date
  9. 查看系统版本
    cat /etc/redhat-release
  10. 查看系统内核
    cat /proc/version
  11. 查看系统加载模块
    lsmod
  12. 查看系统加载环境变量
    env

# 文件权限

  1. 查看文件权限
    ls -l
  2. 修改文件权限
    chmod 777 文件名
  3. 修改文件所属用户
    chown 用户名 文件名
  4. 修改文件所属用户组
    chgrp 用户组 文件名

# 文件搜索

  1. 搜索文件
    find /-name 文件名
  2. 搜索文件内容
    grep “关键字” 文件名
  3. 搜索文件内容
    grep -r “关键字” 目录名
  4. 搜索文件大小
    find /-size +100M
  5. 搜索文件时间
    find /-mtime +10
  6. 搜索文件类型
    find /-type d
  7. 搜索文件权限
    find /-perm 777
